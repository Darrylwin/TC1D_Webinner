Historique du HTML

HTML 1.0 (1993) : La première version d’HTML a jeté les bases de la construction de pages web. Elle a permis d’écrire des documents hypertextuels liant les différentes ressources d’Internet avec des hyperliens.

HTML 2.0 (1995) : Elle a notamment introduit les tables pour les données tabulaires.

HTML 3.2 (1997) : Publiée en tant que recommandation du W3C, cette version a continué à élargir la gamme d’éléments pris en charge.

HTML 4.0 (1997) : Une autre recommandation du W3C, cette version a introduit de nouvelles fonctionnalités telles que les formulaires et les cadres.

HTML5 (à partir des années 2010) : HTML5 est la version la plus récente et la plus utilisée. Elle offre des fonctionnalités avancées pour structurer sémantiquement les pages web, inclure des ressources multimédias et créer des documents interopérables.

La version actuelle de HTML est HTML5


Qui est le créateur du Web ?

Le World Wide Web (WWW), plus communément appelé le Web, a été inventé par Tim Berners-Lee, un physicien et informaticien britannique.
Il est un informaticien britannique né le 8 juin 1955 à Londres.
Il a proposé un système de gestion de l’information le 12 mars 1989 et a mis en œuvre la première communication réussie entre un client et un serveur HTTP (Hypertext Transfer Protocol) via Internet.
Il est également le fondateur et directeur du World Wide Web Consortium (W3C), qui supervise le développement continu du Web.
Tim Berners-Lee a reçu de nombreuses distinctions pour son invention, dont le Prix Turing en 2016 pour avoir inventé le World Wide Web, le premier navigateur Web et les protocoles et algorithmes fondamentaux permettant au Web de se développer à grande échelle.

Historique du CSS

CSS1 (1996) : La première spécification officielle pour CSS a été publiée en décembre 1996. Elle a introduit des fonctionnalités de base pour la mise en forme du texte, les propriétés de police, les marges, les rembourrages, les couleurs et les arrière-plans. Le modèle de boîte (box model) a également été défini dans CSS1.

CSS2 (1998) : Publiée en mai 1998, CSS2 a étendu CSS1 avec des sélecteurs plus sophistiqués, des propriétés de positionnement, des feuilles de style spécifiques aux médias, etc.

CSS2.1 : Une révision importante qui a corrigé les incohérences et les erreurs trouvées dans CSS2. Elle est devenue une recommandation du W3C en juin 2011 et est largement prise en charge par tous les navigateurs.

CSS3 : Contrairement à une spécification monolithique, CSS3 est divisée en plusieurs modules indépendants. Ces modules incluent des fonctionnalités telles que les sélecteurs, le modèle de boîte, les arrière-plans, les effets de texte, les transformations 2D/3D, les animations, Flexbox, Grid Layout, etc. CSS3 a permis des designs plus réactifs, des mises en page avancées et des améliorations visuelles.

CSS Living Standard : CSS est considéré comme un standard vivant, avec de nouvelles fonctionnalités et modules développés au fil du temps. Le groupe de travail CSS du W3C continue de rédiger et d’affiner les spécifications CSS.

La version actuelle de CSS est CSS3

Qui est le créateur du CSS ?

Le CSS (Cascading Style Sheets) est un langage de feuille de style utilisé pour définir la présentation et le style des documents HTML. Il a été créé par Håkon Wium Lie et Bert Bos en 1996. Le CSS permet de contrôler la mise en forme des éléments HTML tels que les polices, les couleurs, les marges, les bordures et bien plus encore.
